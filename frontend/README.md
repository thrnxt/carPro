# Frontend - Car Maintenance Platform

Frontend приложение для платформы учета обслуживания автомобилей.

## Технологии

- React 18
- TypeScript
- Vite
- React Router
- TanStack Query (React Query)
- Zustand (State Management)
- Tailwind CSS
- Axios

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно на `http://localhost:3000`

## Сборка

```bash
npm run build
```

## Структура проекта

```
src/
├── api/           # API клиент и конфигурация
├── components/    # Переиспользуемые компоненты
├── pages/         # Страницы приложения
├── store/         # Zustand stores
└── App.tsx        # Главный компонент
```
